# Support

Thanks for using cfgable! Here is where to go for help.

## Questions and usage help

- Read the [README](README.md) — it covers installation, the core idea, the Hydra
  bridge, and round-tripping.
- Browse [existing issues](https://github.com/OpenGHz/cfgable/issues) to see if your
  question has already been answered.
- Open a [discussion or question issue](https://github.com/OpenGHz/cfgable/issues/new/choose)
  if you can't find an answer.

## Reporting bugs

Open a [bug report](https://github.com/OpenGHz/cfgable/issues/new/choose) with a
minimal reproducible example, the cfgable version, and your Python version.

## Requesting features

Open a [feature request](https://github.com/OpenGHz/cfgable/issues/new/choose)
describing the use case and the API you have in mind.

## Security issues

Please follow the process in [SECURITY.md](SECURITY.md) — do not file security
reports as public issues.
